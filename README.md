# PDF Sentence Navigator

Chrome Extension, რომელიც PDF დოკუმენტში წინადადება-წინადადებით
გადაადგილების საშუალებას იძლევა კლავიატურით:

- **Tab** — შემდეგი წინადადება
- **Shift + Tab** — წინა წინადადება

აქტიური წინადადება ყვითლად მოინიშნება PDF-ში.

## ჩატვირთვა Chrome-ში

1. გახსენი `chrome://extensions`
2. ჩართე **Developer mode** (ზედა მარჯვენა კუთხეში)
3. დააჭირე **Load unpacked**-ს და აირჩიე ეს ფოლდერი
4. Extension გამოჩნდება toolbar-ზე (თუ არ ჩანს — puzzle ხატულაზე დააჭირე და დააპინე)

## გამოყენება

**PDF ფაილიდან:**

1. დააჭირე extension-ის ხატულას → გაიხსნება viewer
2. აირჩიე PDF ფაილი კომპიუტერიდან
3. დააწკაპუნე დოკუმენტზე და დააჭირე **Tab** / **Shift+Tab**

**ვებ-გვერდზე PDF ბმულიდან:**

1. მარჯვენა წკაპი PDF ბმულზე
2. აირჩიე **"PDF-ის გახსნა Sentence Navigator-ში"**
3. დააჭირე **Tab** / **Shift+Tab**
